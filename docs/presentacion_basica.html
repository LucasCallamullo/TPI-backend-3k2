Reglas de Cálculo de Tarifas

Fórmulas y componentes para calcular los costos de transporte en el sistema

Resumen General del Cálculo

El costo total de la solicitud se obtiene sumando los cuatro componentes principales.

Costo Total =
  Costo Gestión
 + Costo Camión
 + Costo Combustible
 + Costo Estadía

Aplicación: Es la suma final utilizada para determinar el monto total que el cliente debe pagar.

Costo de Gestión

Cargo fijo aplicado por cada tramo de la ruta.

Costo Gestión = cantidad_tramos × costo_gestion_por_tramo
Ruta, Tarifa

Costo del Camión

Costo basado en distancia y costo por km.

Costo Camión = SUM(distancia_km × costo_por_km_camion)
Tramo, Camión

Costo de Combustible

Basado en consumo real por tramo.

Costo Combustible = SUM(distancia_km × (consumo_L_100km / 100) × precio_combustible)
Tramo, Camión, Tarifa

Costo de Estadía

Costo adicional por estadías en depósitos.

Costo Estadía = SUM(costo_estadia_por_tramo)
Tramo/Depósito

Modelo C4

Arquitectura del sistema y descripción de microservicios

Microservicio Clientes

Gestiona clientes e integra con Keycloak para registro seguro.

Endpoints importantes:

  • POST /clientes crear cliente y registrarlo en Keycloak
  • GET /clientes obtener todos los clientes
  • GET /clientes/{keycloackId} obtener información del cliente
Ver docs - Clientes

Microservicio Solicitudes y Contenedores

Administra solicitudes de transporte, contenedores, estados y rutas tentativas vía los microservicios de logistica y routing (OSRM).

Endpoints importantes:

  • POST /solicitudes crear solicitud (cliente)
  • POST /solicitudes/completa crear solicitud y crear usuario y contenedor (admin)
  • PATCH /solicitudes/{id}/asignar-ruta asignar ruta específica
  • PATCH /solicitudes/{id}/calcular-costos-estimados calcula costos estimados para una solicitud indicada
  • PATCH /solicitudes/{id}/calcular-costos-totales calcula costos estimados para una solicitud indicada
  • GET /solicitudes/{id}/rutas-tentativas rutas tentativas
Ver docs - Solicitudes

Microservicio Logística

Gestiona camiones, depósitos, rutas, tramos, tarifas y cálculo de costos operativos. Se comunica con routing-service (OSRM) para el calculo de costos estimados, totales, y calcular distancias.

Endpoints importantes:

  • PATCH /tramos/{id}/asignar-camion/{camionId} asignar camion a un tramo específica
Ver docs - Logística

Microservicio Gateway

Actúa como puerta de entrada única. Aplica seguridad, routing, y filtra tráfico hacia los demás microservicios. Responsable del flujo de OAuth2/OpenID con Keycloak.

Routing Service

Microservicio especializado en obtener distancias reales mediante OSRM (Open Source Routing Machine) usando latitudes y longitudes. Retorna rutas optimizadas con tiempos y distancias precisas que luego son utilizadas por Logística.

Endpoints importantes:

  • GET /calcular-ruta obtiene mediante request params, latitudes y longitudes de origen y destino y calcula la distancia entre ambos puntos

Keycloak Identity Provider

Servicio externo de autenticación centralizado. Maneja login, tokens, usuarios y roles. Provee JWT para seguridad del ecosistema.
